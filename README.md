* [Iris](https://modrinth.com/mod/iris) - необходим для работы шейдеров
* [Jade](https://modrinth.com/mod/jade) - информация о блоке/сущности на который ты смотришь

* [Particle Rain](https://modrinth.com/mod/particle-rain) - мод, который очень круто перерабатывает дождь, снег и т.д.

* [Rolling Down in the Deep](https://modrinth.com/mod/rolling-down-in-the-deep) - улучшенное плавание
* [Do a Barrel Roll](https://modrinth.com/mod/do-a-barrel-roll) + [Forgified Fabric API](https://modrinth.com/mod/forgified-fabric-api) - улучшенный полет на элитрах
